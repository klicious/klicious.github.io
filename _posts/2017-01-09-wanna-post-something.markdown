Syntax Guide

_**Headers**_
# This is an <h1> tag
## This is an <h2> tag
###### This is an <h6> tag

_**Emphasis**_
*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__

_You **can** combine them_

_**Lists**_

**Unordered**
* Item 1

**Ordered**
1. Item 1

3. Item 3

   * Item 3b
